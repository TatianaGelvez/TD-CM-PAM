This repository contains the code related to the paper  
**"Efficient Convolutional Forward Model for Passive Acoustic Mapping and Temporal Monitoring"**

## Getting Started

To run the code, make sure that you:

- Clone this repository
- Install the required dependencies
- Download the **BM4D denoising code**, which is required for the iterative denoising steps and include it in the Support folder.  
  The BM4D denoising code is available here:  
  https://webpages.tuni.fi/foi/GCF-BM3D/#ref_software
